Claves

El subtest Claves es una prueba que evalúa la capacidad del niño o adolescente para detectar patrones y relaciones entre elementos visuales. El subtest se utiliza como parte de la evaluación neuropsicológica y del coeficiente intelectual (CI). Durante el subtest, se presenta al niño o adolescente una serie de imágenes que se organizan en una matriz o cuadrícula. Algunas de las imágenes tienen una relación lógica entre sí, mientras que otras no. El niño o adolescente debe descubrir cuál es la imagen que falta para completar el patrón o relación lógica. La puntuación del subtest se basa en la cantidad de elementos correctamente identificados y en la capacidad del niño o adolescente para detectar patrones y relaciones entre elementos visuales. También se considera la velocidad y la precisión en la ejecución de la tarea. La puntuación del subtest de Claves se utiliza para calcular el índice de razonamiento perceptual en el WISC-IV. Este índice evalúa la capacidad del niño o adolescente para resolver problemas visuales y manipular información visual, y se utiliza como una medida de la capacidad no verbal general del individuo.
